# 시작하며

기술 블로그를 새로 만들었습니다. 이 블로그는 **Next.js**, **Tailwind CSS**, 그리고 **MDX**를 사용하여 구축되었습니다.
개발 과정에서 겪은 경험과 배운 점들을 기록해 나갈 예정입니다.

## 기술 스택 선정 이유

왜 이 스택을 선택했는지 간단히 정리해봅니다.

### 1. Next.js App Router

최신 React 기능을 활용하고, SEO 친화적인 구조를 만들기 위해 선택했습니다. 특히 서버 컴포넌트(RSC)를 통해 성능을 최적화할 수 있습니다.

### 2. Tailwind CSS

스타일링의 자유도와 생산성을 위해 Tailwind를 채택했습니다. 다크 모드 구현도 매우 간편했습니다.

```tsx
// Tailwind 설정 예시
module.exports = {
  darkMode: "class",
  theme: {
    extend: {
      colors: {
        primary: "#3b82f6",
      },
    },
  },
};
```

## 주요 기능 소개

이 블로그에는 몇 가지 특별한 기능이 있습니다.

### Table of Contents (목차)

왼쪽에 보이는 목차는 현재 읽고 있는 위치를 표시해줍니다. `IntersectionObserver`를 사용하여 구현했습니다.

### Code Syntax Highlighting

코드 블럭은 가독성을 위해 스타일링되었으며, 다양한 언어를 지원합니다.
